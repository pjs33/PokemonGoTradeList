How to create the token (one time)
  1. Open github.com/settings/personal-access-tokens/new (logged in as pjs33)
  2. Name it e.g. trade-list-admin, set a long expiration
  3. Repository access → Only select repositories → PokemonGoTradeList
  4. Permissions → Repository permissions → Contents: Read and write
  5. Generate, copy the github_pat_… value, paste it above
